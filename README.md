# Health Scoring System

## Overview
The **Health Scoring System** is a web-based application that allows users to fill out a questionnaire with personal health information. This data is processed through a machine learning model to generate a health score, compare the user’s health status to others, and provide personalized feedback with verified resources based on identified problem areas.

## Purpose
The goal of this project is to help general users better understand their overall health by placing it in context relative to others. By translating raw health inputs into an easy-to-understand score and recommendations, the system aims to increase health awareness and encourage informed lifestyle decisions.

## How It Works
1. Users complete a health questionnaire through the web interface  
2. User inputs are processed by a machine learning model  
3. The model generates a health score and comparison metrics  
4. The system provides feedback and resources based on risk areas

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Node.js, React  
- **Backend:** Python, MySQL, Fast API
- **Machine Learning:** XGBoost, Python, Pandas, Plotly
- **DevOps:** Docker, Terraform (IaC), Bash
- **Cloud Ops:** AWS - VPC, ECR, EC2, RDS, ALB(Application Load Balancer), Route 53, Amplify

## Dataset
The dataset used to train and evaluate the machine learning model was sourced from Kaggle and is provided in CSV format.
- **Source:** Kaggle  
- **Link:** [Global Lifestyle and Lifespan Synthetic Dataset](https://www.kaggle.com/datasets/mabubakrsiddiq/global-lifestyle-and-lifespan-synthetic-dataset)
> Note: The dataset is publicly available and does not contain personally identifiable information.
